# VSSUT SIH 2025 — Grand Finale Deliverables

Official GitHub account for collecting final deliverables from participating teams in the **Smart India Hackathon 2025 Grand Finale** hosted at **Veer Surendra Sai University of Technology (VSSUT), Burla, Sambalpur**.

## Account Structure

- **18 Private Repositories** — one for each participating team
- **Repository Naming** — named after respective team names
- **Access** — one team member has been added as collaborator to their team's repository

## Submission Guidelines

### Accessing Your Repository

The designated team member will receive a collaboration invitation via email. Accept the invitation to access your team's private repository.

### Required Deliverables

Push all deliverables to your team's repository in an organized manner:

1. **Source Code** — complete codebase and configuration files
2. **Documentation** — all relevant project documentation
3. **Presentations** — all presentation files (PPT/PDF)

### Recommended Structure

```
team-name-repo/
├── README.md
├── src/
├── docs/
└── presentations/
```

### Submission Checklist

- Organize files in clearly labeled folders
- Include comprehensive README.md with project details, team members and problem statement ID
- Use clear commit messages
- Remove sensitive information (API keys, passwords)
- Host large files (>100MB) externally with links
